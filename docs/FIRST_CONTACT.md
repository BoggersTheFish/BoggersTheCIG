# First Contact

This is the 90-second version for a skeptical reviewer.

## What This Repo Is

BoggersTheCIG is CIG infrastructure: a local-first provenance-aware concept graph.

It stores claims as graph structure, tracks where those claims came from, attaches heuristic confidence, and helps inspect possible contradictions and bridge nodes.

## What It Helps With

- Store claims and evidence as graph state.
- Inspect provenance for claims.
- Surface possible contradictions.
- Track confidence and reinforcement signals.
- Export graph memory into Obsidian-readable notes.

## What It Is Not

- Not the whole TS system.
- Not a finished autonomous reasoner.
- Not a production knowledge base.
- Not a theorem prover.
- Not proof that heuristic confidence equals truth.

## Where It Fits

Use [TS-Start-Here](https://github.com/BoggersTheFish/TS-Start-Here) for the ecosystem map.

Use [TS-Reasoner-v0](https://github.com/BoggersTheFish/TS-Reasoner-v0) for verifier-backed reasoning traces.

Use this repo when you want to inspect claim/evidence/provenance graph infrastructure.

## Current Status

Active experimental prototype. Useful as CIG infrastructure; limited by extraction errors, heuristic confidence scoring, incomplete contradiction detection, local model dependencies, and noisy vault output.
